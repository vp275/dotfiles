# Archived Linux Configs

This directory preserves the old Linux Stow package for future reference.

Linux is not an active target for this dotfiles repo right now. The active
package is `mac/`, deployed with:

```bash
cd ~/.dotfiles
stow mac
```

The archived Linux files are not part of the active Stow workflow. If Linux
support comes back, review these files against the current `mac/` package before
restoring them.
